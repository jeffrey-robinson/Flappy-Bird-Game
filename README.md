# 🐦 Flappy Bird Game

## 📖 Overview

Welcome to the **Flappy Bird Game** project! This project is a recreation of the classic Flappy Bird game built using **React Native** and **Expo**. The objective is simple: guide the bird safely through the obstacles without colliding with them. Although the gameplay is easy to understand, achieving a high score requires quick reflexes and precise timing.

This project was created to improve my React Native development skills and gain practical experience in building interactive mobile applications. It demonstrates how to manage game logic, user interactions, animations, and responsive layouts using modern React Native concepts.

## 🚀 Getting Started

Follow these steps to run the project on your local machine:

1. Clone the repository.
2. Navigate to the project folder.
3. Install dependencies using `npm install`.
4. Start the Expo development server using `npx expo start`.
5. Scan the QR code with the Expo Go app or run it on an Android Emulator or iOS Simulator.

## 🎮 Gameplay

The player controls the bird by tapping the screen. Each tap makes the bird move upward, while gravity continuously pulls it downward. The challenge is to fly through the gaps between the obstacles without touching them. Every successful pass increases the player's score. The game ends when the bird collides with an obstacle or the ground, after which the player can restart and try again.
